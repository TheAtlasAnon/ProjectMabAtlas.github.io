درباره پروژه

این پروژه یک بازی **Minesweeper** است که با زبان **C++** پیاده‌سازی شده است. بازی شامل یک صفحه مربعی است که در آن بازیکن باید با حدس زدن خانه‌های امن، از مین‌ها دوری کند.

قوانین بازی

  • صفحه بازی به خانه‌هایی تقسیم می‌شود که مین‌ها به طور تصادفی در بعضی از آن‌ها توزیع شده‌اند.
  • بازیکن باید وضعیت تمام خانه‌ها را معلوم کند تا برنده شود.
  • عدد روی هر خانه تعداد مین‌های مجاور آن (8 خانه همسایه) را نشان می‌دهد.
  • با استفاده از این اطلاعات، بازیکن می‌تواند خانه‌های امن را مشخص کند و خانه‌های مین‌دار را با پرچم علامت‌گذاری کند.

ساختارهای مورد نیاز

  • ساختار بازیکن: شامل نام کاربری، تعداد بردها و تعداد باخت‌ها.
  • ساختار زمین بازی: شامل تعداد سطرها، ستون‌ها، مین‌ها و آرایه‌ای از خانه‌ها.
  • آرایه‌ای از بازیکن‌ها: برای ذخیره اطلاعات همه بازیکن‌ها.

منوی بازی

  • شروع بازی جدید: کاربر می‌تواند بین سه سطح دشواری (آسان، متوسط، سخت) انتخاب کند.
  • مشاهده جدول امتیازات: نمایش رتبه‌بندی بازیکن‌ها بر اساس تعداد بردها و باخت‌ها.
  • خروج از بازی: ذخیره اطلاعات بازیکن‌ها در فایل و خروج از برنامه.

ویژگی‌های اضافه

  • نمایش زمان سپری‌شده در بازی.
  • امکان ذخیره و ادامه بازی در آینده.
  • شروع هوشمند: خانه اولی که کاربر انتخاب می‌کند، مین نباشد.
  • امکان استفاده از کلیدهای کیبورد برای حرکت در بازی.
  • ثبت رکوردها و نمایش بهترین زمان‌ها برای هر سطح دشواری.

نمونه کد

در زیر بخشی از کد مربوط به ایجاد صفحه بازی آورده شده است:

                
#include <iostream>
#include <vector>
using namespace std;

const int SIZE = 10;

void initializeBoard(vector<vector<char>> &board) {
    for (int i = 0; i < SIZE; i++) {
        for (int j = 0; j < SIZE; j++) {
            board[i][j] = '-';
        }
    }
}

void printBoard(const vector<vector<char>> &board) {
    for (int i = 0; i < SIZE; i++) {
        for (int j = 0; j < SIZE; j++) {
            cout << board[i][j] << " ";
        }
        cout << endl;
    }
}

int main() {
    vector<vector<char>> board(SIZE, vector<char>(SIZE));
    initializeBoard(board);
    printBoard(board);
    return 0;
}
